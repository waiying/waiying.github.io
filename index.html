Projects

Tilt the Snake (Personal)

Tilt the Snake is a mobile game I made for both Android and iOS over the summer of 2016. It is a game where the player tilts his/her device to control the snake to eat apples for points and avoid any obstacles in the way. I developed the whole game except for the sound effect and background music.

Technologies/API(s): Unity, Photoshop CC, XCode, Facebook API, Admob API

Tilt the Snake on GitHub
Tilt the Snake on Google Play

Pong (Personal)

Pong is my first mobile game made for Android during the summer of 2016. It is a classic Pong game where the player can play against an AI or against a friend. This game uses a multitouch feature for multiplayer on the same device. The player can choose the difficulty of the gameplay as well as the selection of background music.

Technologies: Unity, Photoshop CC

Pong on GitHub

Cliffhanger (Team)

Cliffhanger is a game made in a team of four during a eight hours Game Jam at UC Davis. Cliffhanger follows the Game Jam's theme, "maximum power", where the player uses just the right amount of power to throw rocks at the hanging stickman to prevent him from gaining full power to climb up the cliff. I coded and designed the mechanism for the throwing of projectiles as well as the power meter for the strength of the throw.

Technologies: Unity, Photoshop CC

Cliffhanger on GitHub

Text-based Minesweeper (School project)

This is a text-based minesweeper where the player can specify the grid size and the number of mines to play. Mines are generated at different places on the grid randomly. The game is to be ran in command line. This project makes use of recursion to reveal all the tiles without mines surrounding the player's chosen tile.It also checks for the understanding of structs in C programming by using them to keep track of the grid's and each tile's current state.

Language: C
